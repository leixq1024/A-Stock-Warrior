# A-Stock-Warrior v2.0 开发文档

## 当前功能

- ETF波段心法策略分析
- 支持命令行输入多个ETF代码
- 计算技术指标：MA均线、KDJ
- 筛选符合买入条件的ETF

## 使用方式

```bash
node src/index.js 510300 512880 515120 588000
```

## 策略配置

修改 `src/config/index.js` 调整策略参数。

## 数据API

- 实时行情: 新浪财经 `hq.sinajs.cn`
- K线数据: 腾讯财经 `web.ifzq.gtimg.cn`
